# かんなの自律日記システム - 実装計画

## 概要
かんなの自律日記システムを完全実装するための計画。

## Phase 1: データ収集モジュール（4タスク）

### Task 1: Discord API連携 - メッセージ収集機能
- Discordチャンネルから当日のメッセージを収集
- リアクションも含めて収集
- メッセージをタイムスタンプと共に保存
- 期待時間: 30分

### Task 2: GitHub API連携 - Issues/PR収集機能
- GitHub Issues/PRを収集
- 更新日時でフィルタリング（当日のみ）
- コメントも含めて収集
- 期待時間: 30分

### Task 3: カレンダー連携 - Google Calendar連携
- Google Calendar APIを使用して当日のイベントを収集
- 予定のタイトル、時間、場所を取得
- 期待時間: 30分

### Task 4: Obsidian連携 - Daily Notes読み取り機能
- Obsidian VaultからDaily Notesを読み取り
- Markdownパースして内容を抽出
- タスク、メモを分類
- 期待時間: 30分

## Phase 2: データ処理モジュール（2タスク）

### Task 5: Collector実装 - データ集約・フィルタリング
- 各ソースから収集したデータを集約
- 重複排除
- 重要度スコアリング
- 関連情報のグループ化
- 期待時間: 45分

### Task 6: 感情分析モジュール - 感情解析機能
- テキストから感情を抽出
- 感情カテゴリ（喜び、悲しみ、驚き、怒り、学びなど）
- 感情タイムラインの生成
- 期待時間: 45分

## Phase 3: 日記生成モジュール（2タスク）

### Task 7: Generator実装 - AI日記生成エンジン
- OpenAI API (gpt-4o-mini) を使用
- かんならしいパーソナリティで日記生成
- 振り返り、気分、学び、目標を含む日記を生成
- 期待時間: 60分

### Task 8: Writer実装 - 日記保存・Discord投稿
- 生成した日記をMarkdownファイルに保存
- Discordチャンネルに投稿（オプション）
- 日記管理機能（保存、読み取り）
- 期待時間: 45分

## Phase 4: 統計・検索機能（2タスク）

### Task 9: 統計モジュール - 感情統計・活動パターン分析
- 感情の傾向分析
- 活動パターンの分析
- 学びの傾向分析
- 統計情報をグラフ化（オプション）
- 期待時間: 45分

### Task 10: 検索モジュール - 日記検索機能
- キーワード検索
- 日付範囲検索
- 感情フィルタリング
- 期待時間: 30分

## 合計
- タスク数: 10個
- 総期待時間: 6時間
- 並列開発（3エージェント）で推定2時間
